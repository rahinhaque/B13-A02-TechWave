# TechWave Podcast Website

TechWave is a modern, responsive website designed for a tech-focused podcast. It features a sleek, dark-themed UI with vibrant gradients and interactive elements, built to showcase podcast episodes, host information, and community statistics.

![TechWave Preview](https://rahinhaque.github.io/B13-A02-TechWave/)

## Features

- **Responsive Design**: Fully adaptable layout that looks great on desktop, tablet, and mobile devices.
- **Modern UI**: Dark mode aesthetic (`#1a0b2e`) with custom gradients and waveform graphics.
- **Interactive Elements**: Smooth hover effects on cards, buttons, and social links.
- **Key Sections**:
  - **Hero**: Engaging landing area with "Listen on Spotify" and "Subscribe" CTAs.
  - **About**: Brief overview of the podcast's mission and audience.
  - **Stats**: Key performance metrics (Listeners, Episodes, Ratings).
  - **Features**: Bento-grid style layout highlighting podcast benefits (Premium Audio, Global Community, etc.).
  - **Featured Episodes**: Video thumbnails with click-to-play functionality.
  - **Meet The Host**: Host profile with bio and social media links.
  - **Footer**: Custom SVG waveform visuals and platform links.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling using Flexbox, CSS Grid, and CSS Variables.
- **Font Awesome**: Scalable vector icons for UI elements and social media.
- **Google Fonts**: "Inter" font family for clean, modern typography.
